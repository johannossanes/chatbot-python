# CHAT BOT GEMINI

Um chatbot simples desenvolvido com Python, Flet e Gemini 

## Como rodar

### Clone o repositório: 
git clone https://github.com/johannossanes/chatbot-python

### Instale as dependências:

pip install google-generativeai
pip instlall flet 

### Substitua:
"SUA_API" pela sua API_KEY do Gemini 

### Execute: 
python chatbot.py

E aproveite :)
